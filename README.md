# Fashion Retail Sales Analytics Dashboard

## Overview

This Power BI dashboard analyzes over 3,400 retail transactions to provide executive insights into sales performance, customer behavior, product trends, and payment methods.

## Features

- Data cleaning with Power Query
- Star schema data model
- DAX measures
- KPI cards
- Interactive slicers
- Drill-through navigation

## KPIs

- Total Sales
- Average Order Value
- Total Orders
- Unique Customers
- Average Rating

## Tools Used

- Power BI
- Power Query
- DAX
- Excel
